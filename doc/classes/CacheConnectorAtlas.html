CacheConnectorAtlas Class

Defined in: connector/atlas/CacheConnectorAtlas.js:16
Module: connector.Atlas

CacheConnectorAtlas is the cache level dedicated to the atlas data-api for DNSMON. It provides a bottom-up incremental cache and top-down decremental queries.
