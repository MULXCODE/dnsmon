SessionManager Class

Defined in: session/SessionManager.js:10
Module: session

SessionManager is the layer providing all the functions to manage the session. It provides a transparent layer for the tool to store and retrieve parameters. The persistence can be provided (or not) by cookies or with any other possible server interaction.
