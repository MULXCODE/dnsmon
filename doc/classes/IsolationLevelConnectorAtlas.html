IsolationLevelConnectorAtlas Class

Defined in: connector/atlas/IsolationLevelConnectorAtlas.js:24
Module: connector.Atlas

IsolationLevelConnectorAtlas has a fundamental role providing isolation between the internal input format and the external input format. It allows to block the propagation of changes affecting the JSON format provided by the data-api. In addition it is a transposition level between a format convenient for the data-api (e.g. light data data format) and a format convenient for the tool (e.g. less computation format).
